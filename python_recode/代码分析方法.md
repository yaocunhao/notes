- 编写方法

  ```python
  import cProfile, pstats
  
  pr = cProfile.profile()
  pr.enable()
  # do something
  pr.disable()
  
  # 对结果进行分析
  stats = pstats.Stats(pr)
  stats.sort_stats(pstats.SorKey.TIME) # 排序
  stats.print_stats()
  # stats.dump_stats(filename='') 写入到文件之中
  
  snakeviz filename # 可视化分析结果
  ```
