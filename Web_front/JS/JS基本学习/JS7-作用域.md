# 二、变量的作用域

- 在函数内部美哦与声明直接复制的变量也是全局变量
- 函数的形参也可以看做局部变量
- 从执行效率来看全局变量和全局变量
  - 全局变量只有游览器关闭的时候才会销毁，比较占内存资源
  - 局部变量，当执行完就会销毁，节省资源
