# 一、什么是变量

- 变量就是一个盒子，用于存放数据的容器

  ```html
  <!DOCTYPE html>
  <html lang="en">
  
  <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script>
      // 声明一个age变量
      var age;
      // 赋值
      age = 19;
      // 输出
      console.log(age)
  
      // 变量的初始化
      var myname='lihua'
      console.log(myname)
  
      </script>
  </head>
  
  <body>
  </body>
  
  </html>
  ```
