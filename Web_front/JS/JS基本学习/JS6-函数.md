# 一、函数的定义和调用

## 1.1 基本介绍

```javascript
// 函数声明关键字 function
function getSum(num1, num2) {
	console.log(num1)
  
  // 只可以返回一个值
  // 如果函数没有返回值则返回undefined
  return num1; 
}

// 函数调用
getSum(1,2);
```
